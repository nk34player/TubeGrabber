# TubeGrabber
is a youtube video/audio downloader made in vb.net

# Features
- Download videos in different extensions (MP3,MP4,WEBM)
- Download at high speed
- Easy & Simple UI
- Updated weekly

# Upcoming features
- Download many videos at one time
- Add custom theme
- Updater
